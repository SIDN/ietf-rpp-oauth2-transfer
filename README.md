# Secure Object Transfer for RESTful Provisioning Protocol (RPP)

This repository contains the IETF draft for securing object transfers in the RESTful Provisioning Protocol using OAuth 2.0.

The [draft](https://github.com/SIDN/ietf-rpp-oauth2-transfer/blob/main/src/draft-wullink-rpp-oauth2-transfer.md) is authored using [mmark](https://mmark.miek.nl/) Markdown.

Contributions in the form of a Pull Request are welcome.

For generated output see:   
[Plaintext](https://sidn.github.io/ietf-rpp-oauth2-transfer/draft-wullink-rpp-oauth2-transfer.txt)  
[HTML](https://sidn.github.io/ietf-rpp-oauth2-transfer/draft-wullink-rpp-oauth2-transfer.html)  
[PDF](https://sidn.github.io/ietf-rpp-oauth2-transfer/draft-wullink-rpp-oauth2-transfer.pdf)  
[XML](https://sidn.github.io/ietf-rpp-oauth2-transfer/draft-wullink-rpp-oauth2-transfer.xml)

## Command Line Usage

Requirements:

- [mmark](https://mmark.miek.nl/)
- [xml2rfc](https://github.com/ietf-tools/xml2rfc#installation)

Formatted versions can be built using:

```sh
cd src
make all
```
